# Weather API Project

A simple **Weather Application** that fetches real-time weather data using a public Weather API and displays it based on the city entered by the user.
This project demonstrates how to **work with APIs, fetch data using JavaScript, and display dynamic weather information in the UI**.


# Features

* 🌍 Search weather by city name
* 🌡 Shows current temperature
* ☁ Displays weather condition
* 💨 Shows wind speed / humidity (if available)
* ⚡ Real-time API data fetching
* 📱 Simple and responsive UI


## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**
* **Weather API**
* **Fetch API**

⚙️ How It Works

1. User enters a **city name**
2. JavaScript sends a request to the **Weather API**
3. API returns weather data in **JSON format**
4. The application displays:

* Temperature
* Weather condition
* Location
* Weather icon


## 📥 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/git-rajudas/WeatherAPI-Project.git
```

Go to project folder:

```bash
cd WeatherAPI-Project
```

Open `index.html` in your browser.

---

## 🔑 API Setup

1. Get a free API key from a weather service (like WeatherAPI or OpenWeather).
2. Replace the API key in `script.js`.

Example:

```javascript
const API_KEY = "your_api_key_here";
```
